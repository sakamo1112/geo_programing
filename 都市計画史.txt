3. 都市計画のあるべき姿
景観条例とアーバンデザインの話について。赤: 206
まちづくり条例が都市個性を生むわけではないという話。
総合的な視点の持ち方と、計画の中で位置付けられる市民参加。

市民参加の形

行政の役割は、都市を生きている存在として捉え、その中で個性が磨かれ育てられ、
時として新たな個性が生まれるようなコミュニティの育成を支援することである。
そのためにも、住む人々が自分の都市に愛着を持てることが重要である。都市を構成する
地域の中には様々な主体が存在し、それらは地域に愛着を持つ人々によって運営されるからである。
また現代においては多くの都市において人口減少が進みこれまでは行政が担ってきた役割を
担いきれなくなっている地域の存在や、新たに生まれてきた課題も存在している。
例えば高齢化・人口減少の進行による住民の孤立などである。また、災害時の避難所運営は
行政の人員だけではカバーできないことも少なくなく、住民の自主的な活動が必要となる。
特に「令和6年能登半島地震」で生じたような、自治体単位での孤立状態においては
住民内の結びつきが一時的な避難所運営のみならず、長期的な復興においても重要となるだろう。

つまり、個性的な都市をつくっていくためには
・市民意識の育成
・市民意識を実際の都市づくりに活かせる「シクミ」づくり
が必要であり、これらを実現する活動の繰り返しの中で、2つの要素は相互に影響し合い
育っていくものであると考える。


```
個性的で魅力的な都市にするためには、ただ狭い意味での観光収入だけを目的にするのではなく、
本当に自分たちの都市をよくすることにつなげ、居住者にとって住みよい個性的な町をつくってゆくことに
つなげなければならない。
```

比較ができないということは、どちらもわかっていないということ。

```
...全国画一的なマニュアルをつくる。これによってできてくる「モノ」は、個性的にはならない。
だが、「ヒト」が結集して十分地域の知恵を発揮できる「シクミ」になっていれば、
かなりおもしろい「モノ」が生まれてくるだろう。
```
目にみえ、外見から分かりやすい「モノ」や「コト」を真似るだけでなく、目に見えない「シクミ」や市民の精神の中に、
その都市"らしさ"が育つよう努めることが必要ではないだろうか。
そして個性的な「シクミ」の中で育まれた市民意識の中から出てきた「モノ」を掬い認める。このようなプロセスの中から、
新たな個性は生まれてくるのではないか。

しかし、田村氏は⭕️⭕️の中で、「個性的な都市にするにはどうするのがよいか」と聞かれた際には、
```
都市を個性化し、都市を美しくするマニュアルをつくるということはありえない。
それじたいが個性化に反することである。
```
と説明していると述べている。その上で、個性化を求めるためには、その方法から
すでに個性的でなければならないと述べている。
つまり、誰かが教えることによって型にはめることは個性化へ向けたアプローチとして
適切ではなく、他都市を参考にしながらも、あくまで自分たちで考えていくべき問題
であると述べている。ここで大事なことは「主体性を失わない」ということである。

問題として、都市をつくっている主体の多くは自分たちのことには関心があるものの
都市全体の話となると関心がないことがしばしばある点が挙げられる。
この問題に関しては、各主体が「自分たちも都市をつくっている一部である」ということである
意識をいかに持たせるかが重要である。
多数の主体が連携し共通の目的を達成するための取り組みを実行に移すことのできる「シクミ」が必要なのである。
そしてその基礎となるものが地域の共同体であり、「自治体」は地域共同体の最も典型的で
法的にも制度化された「自治体」が積極的にになっていくべき役割ではないだろうか。
都市計画マスタープランに定められるような計画は、都市として目指す方向について、
長期的・広域的な視点からある程度の方向性を示すものであって、都市の中に存在する
様々な主体の活動を認め計画の中に位置付けることが必要である。したがって、
マスタープランは都市の目指すべき将来像を勝手に描き示すものではなく、自治体の
中での足並みを揃えるためのプランであるとともに、市民が「自分たちの手で都市を
つくっている」という意識を持つきっかけになるためのプランであるべきでは
ないだろうか。


「らしさ」の熟成=都市における市民の主体性の育成->自助的活動の育成
